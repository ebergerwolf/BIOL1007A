Glomeruli are a way to section groups of neurons. Neurons of the same glomerular class typically have similar structure, function, and/or location. We can look at the different glomerular classes of the neurons in this dataset.

summary(dros_data[,"Glomerulus"]) #number of each neurons in each glomerular class
##  DA1  DL3 DP1m VA1d 
##   11   10    8   11

Lets say we want to see if there is a difference in the number of segments between neurons in different glomeruli. We can start by extracting the number of segments for each neuron to see if they do generally differ.

segment_data <- data.frame(neuron_ID = c(1:length(dros_data)), num_segs = rep(NA, length(dros_data)))
for(i in 1:length(dros_data)){
  segment_data[i,"num_segs"] <- dros_data[[i]]$NumSegs
}

hist(segment_data$num_segs, breaks=20)

It looks like we have a generally bimodal distribution of the number of segments. Lets see if it significantly differs between glomeruli.

segment_data$glomerulus <- dros_data[,"Glomerulus"]
ggplot(segment_data, aes(x=glomerulus, y=num_segs, fill=glomerulus)) +
  geom_boxplot() +
  theme(legend.position="none")

At first glance, DP1m neurons seem to have a significantly different number of segments than the other glomeruli. Lets double check this with an ANOVA.

segment_data$glomerulus <- factor(segment_data$glomerulus)
glom_aov <- aov(num_segs ~ glomerulus, data=segment_data)
summary(glom_aov)
##             Df Sum Sq Mean Sq F value Pr(>F)

Now that we know that there is at least one signifcantly different glomerulus, lets conduct a post-hoc test to determine which of the glomeruli significantly differ in segment number.

TukeyHSD(glom_aov)

Here, we can confirm that it is the DP1m glomerulus that has a significantly different number of segments. Lets take a look at the morphology of the DP1m neurons versus the rest.

plot(dros_data, subset=Glomerulus!="DP1m", col='red', 
     WithNodes=F, main="DP1m vs. all other neurons") 
plot(dros_data, subset=Glomerulus=="DP1m", add=TRUE, col="blue", WithNodes=F)
legend('bottomright', legend = c("Not DP1m","DP1m"), fill=c("red","blue"))

Now we can see the qualitative difference in the number of segments of DP1m neurons from non-DP1m neurons. And thats one example of what the nat package can be used for!
